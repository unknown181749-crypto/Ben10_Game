## 🎮 Features

### Core Gameplay
| Feature | Description |
|---------|-------------|
| 🕹️ **Third-Person Action** | Smooth camera following and intuitive controls |
| ⚔️ **Combat System** | Melee attacks with visual feedback and animations |
| 🔥 **Combo System** | Chain attacks for score multipliers |
| ❤️ **Health System** | Dynamic health bar with color indicators |
| 🏆 **Score Tracking** | Earn points for hits and defeating enemies |

### Game Modes
| Mode | Description |
|------|-------------|
| ▶️ **Play Game** | Full player-controlled gameplay |
| 🎬 **Watch Demo** | AI plays automatically - great for showcasing |
| 📖 **Tutorial** | Step-by-step interactive guide |

### Visual Style
- ✨ **Low-Poly 3D Art** - Clean, stylized graphics
- 🎨 **Ben 10 Theme** - Signature green/black color scheme
- 🌲 **Rich Environment** - Trees, rocks, cave, van, watchtower
- 💫 **Smooth Animations** - Walk cycles, attack animations, death effects

## 📦 Installation

### Prerequisites
- **Python 3.7** or higher
- **pip** package manager

### Quick Start

```bash
# 1. Clone or download the project
git clone <repository-url>
cd ben10-alien-force

# 2. (Optional) Create virtual environment
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

# 3. Install dependencies
pip install ursina

# 4. Run the game
python ben10_alien_force_demo_game.py
```
